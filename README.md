# Excel Data Analytics & Dashboard Project

This repository contains an end-to-end Excel project completed during a hands-on data analytics tutorial. The project covers the full workflow of cleaning, analyzing, and visualizing data using Excel’s advanced features.

# Project Overview
# 1. Data Cleaning
Handling missing values<br>
Removing duplicates<br>
Standardizing formats<br>
Splitting/combining fields<br>
Applying filters to isolate issues

# 2. Formulas & Functions
XLOOKUP & VLOOKUP<br>
IF statements<br>
TEXT functions (LEFT, RIGHT, MID, TRIM)<br>
Date & time functions<br>
Aggregation formulas (SUM, COUNT, AVERAGE, etc.)

# 3. Data Analysis Tools
PivotTables<br>
PivotCharts<br>
Dynamic filtering using Slicers & Timelines

# 4. Conditional Formatting
Highlighting patterns and trends<br>
Color scales for performance metrics<br>
Custom rules for flagging important insights

# 5. Interactive Dashboard
A clean and interactive Excel dashboard designed to:<br>
Visualize KPIs<br>
Explore insights using slicers<br>
Present trends and summaries for decision-making

# How to Use This Project
Download the Excel files.<br>
Open in Microsoft Excel (2016 or later).<br>
Go through the raw data, cleaned data, and final dashboard.<br>
Adjust slicers to explore the dashboard interactively.
